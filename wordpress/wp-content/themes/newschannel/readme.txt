===========
ABOUT Theme
===========
It's a fun and functional theme perfect for the personal blogger. Featuring a gallery-style layout it supports custom menus and is widget ready.

This theme is compatible with Wordpress Version 3.4 and above and it supports the new custom background feature (https://codex.wordpress.org/Custom_Backgrounds).

IMPORTANT: This theme is designed with black/dark background in mind. So if you decide to change background image or background color, make sure that it is dark enough to see the white fonts in the header, comments area and footer.

All the image graphics and icons bundled with this theme are created by the author and licensed under the GPL.

For free themes support, please contact us http://citizenjournal.net/contact/
