== Description ==
Collection of widgets for bbPress plugin powered forums. Currently included widgets:

* Search Topics (requires GD bbPress Tools plugin)
* Topics Views (selectable list of views to display)

Supported languages: English.

== Installation ==
= General Requirements =
* PHP: 5.2.x
* bbPress: 2.x.x

= WordPress Requirements =
* WordPress: 3.2

= bbPress Requirements =
* bbPress Plugin: 2.0 or newer

= Basic Installation =
* Plugin folder in the WordPress plugins folder must be `gd-bbpress-widgets`
* Upload folder `gd-bbpress-widgets` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
* Where can I configure the plugin?
Open the Forums menu, and you will see Widgets item there. This will open a panel with global plugin settings.

* Will this plugin work with standalone bbPress instalation?
No. This plugin requires the plugin versions of bbPress 2.0 or higher.

* Does this plugin work with bbPress that is part of BuddyPress plugin?
No. Plugin requires bbPress 2.0 or higher plugin.

* Does this plugin work with bbPress 2.1 used as site wide forums for BuddyPress plugin?
Yes. But, make sure to enable 'Always Include' option for JavaScript and CSS.
